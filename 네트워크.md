# 체육복
> https://school.programmers.co.kr/learn/courses/30/lessons/43162
- 사용언어 : JAVA

## 문제풀이
1. DFS를 사용해 풀었다.
2. 네트워크의 갯수는 탐색한 DFS의 수이다.

## Code1
```
import java.util.*;

class Solution {
    List<Node> nodeList;
    public int solution(int n, int[][] computers) {
        nodeList = new ArrayList<>();
        for(int i = 0; i<computers.length; i++){
            Node node = new Node(i);
            nodeList.add(node);
            for(int j = 0; j < computers[i].length; j++){
                if(i == j){
                    continue;
                }
                if(computers[i][j] != 0){
                    node.side.add(j);
                }
            }
        }
        int answer = 0;
        for(int i =0;i<n;i++){
            Node node = nodeList.get(i);
            if(!node.marked){
                answer++;
                dfs(node);
            }
        }
        return answer;
    }
    
    class Node{
        int data;
        boolean marked;
        List<Integer> side;
        
        Node(int data){
            this.data = data;
            side = new ArrayList<>();
        }
    }
    
    public void dfs(Node node){
        if(!node.marked){
            node.marked = true;
            for(int n : node.side){
                dfs(nodeList.get(n));
            }
        }
    }
}
```
